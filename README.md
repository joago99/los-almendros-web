# Constructora Los Almendros — Sitio web

Landing page estática para la constructora, hospedada en GitHub Pages.

## Stack

- HTML + CSS vanilla
- Tailwind CDN
- Activos en `assets/`
- Formulario: Google Forms

## Status

- Estado: **production**
- URL: https://constructoralosalmendros.cl
- Hosting: GitHub Pages + Cloudflare DNS
- Repo: https://github.com/joago99/los-almendros-web

## Secciones principales

- Hero con carrusel
- Sobre Nosotros (foto residencial, no nombres)
- Servicios
- Proyectos
- Cotización
- Contacto

## Convenciones

- No hardcodear información personal del equipo en “Sobre Nosotros”
- Links y textos deben coincidir con `llms.txt` y `sitemap.xml` cuando corresponda
